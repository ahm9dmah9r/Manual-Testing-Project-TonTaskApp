# Ton Task App

## Description
Ton Task is a mobile application designed to help users manage personal tasks, whether they are one-time or recurring. The app also includes social collaboration features, allowing users to create public tasks, interact with other users through likes and comments, and follow each other’s activity.
The app supports both English and Arabic languages and provides multiple login options (Email, Google, Apple). It focuses on task management, personalization, and smooth user experience.

## Features
    • User Registration and Login
        ◦ Email/password login
        ◦ Google and Apple authentication
    • Task Management
        ◦ Create one-time tasks
        ◦ Create repeatable tasks (weekly, monthly, yearly)
        ◦ View all tasks on the dashboard
    • Account Settings
        ◦ Change name, password, and profile picture
        ◦ Change app language between English and Arabic
    • Public Task Interaction
        ◦ Add, share, like, and comment on public tasks
        ◦ Follow/unfollow other users
    • Secure Logout and session management
    
  ##  Goals
    The goal is to provide a mobile-first solution for efficient personal task tracking and social engagement. Users can not only organize their lives through scheduled tasks, but also collaborate and engage with a broader community by interacting with public tasks.


### User Account Management
- User Registration with validation and error handling
- Login via email/password, Google, or Apple accounts
- Change account name, password, and profile image
- Logout securely
- Multi-language support: switch between English and Arabic

### Task Management
- Create one-time tasks with title, date, and optional description
- Create repeatable tasks: weekly, monthly, or yearly with customizable repeat intervals
- Add public tasks visible to all users
- Share public tasks with specific users or groups
- Like and comment on public tasks
- Follow other users to stay updated on their activities

## Usage
1. Register an account or log in using email/password or social accounts (Google/Apple).  
2. Access the dashboard to create, view, and manage tasks.  
3. Choose task type: one-time or repeatable (weekly, monthly, yearly).  
4. Add task details: title, date/repeat interval, and optional description.  
5. Manage account settings such as profile name, password, image, and language preferences.  
6. Explore public tasks, interact with others by liking, commenting, sharing, and following users.  
7. Log out when done to secure your account.

## Requirements
- Compatible with Android and iOS devices  
- Internet connection for account authentication and data syncing
